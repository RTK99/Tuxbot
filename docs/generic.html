Feautures

These features are included with Tuxbot!

- Kick/ban

- Userinfo/serverinfo

- 8ball

- Fortune teller

- fetchavatar

- eval(reserved for me, kek)

- distro- get a custom distro tag on your nickname

- YouTube Music: play, add, queue, join, musichelp

And many more to come! Just run ./help to get an updated list.

If you'd like to add a feature of your own, all you have to do is go to the github page, and open a pull request! I am open to merging pull requests that you may have.
